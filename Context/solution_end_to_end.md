# Solution 4: Guided AI Onboarder

### **Concept**
A sophisticated AI consultant that interviews the fleet manager to dynamically build custom inspection forms. It provides intelligent, context-aware guidance for editing existing drivers and creating new ones. All training materials are auto-generated and placed in a centralized hub for a manager-controlled rollout.


### **User Story Map**

*   **Goal:** I want to set up my vehicle inspection process quickly and correctly with expert guidance.
	* Activity 1: Engage with the agent to setup DVIR:
		* Task: Land on Inspection History Page
		* Task: Click on button Setup DVIR with Reveal Intelligence
		* Task: When the user engages with the AI agent the ai agent will present itself and mention the tasks below that he can do on user's behalf   
    * **Activity 2 : Build the right inspection form**
        *   *Task:* Choose how to create the Inspection form:
	        * Build with AI
	        * I have an existing inspection form I want to use
        * *Task:* When user selects build with AI selected: Answer 3 questions: 
	        * What type of vehicles: Light commercial vehicles, Heavy Trucks, Specialized vehicles (emergency, construction, special transport)
	        * Do you need to inspect trailers: yes/no
	        * What is your main priority for vehicle inspections: compliance focused, safety first, maintenance & uptime
	      - Task: When user selects I have an existing inspection form I want to use: 
		      - Present option to upload PDF or Image
        *  Task: Review the form generated by AI 
	        * AI presents loading state while the form is being generated 
	        * When the form is ready AI presents Mobile preview of the Form so the user can see if it's what he needs (default layout is Mark defects only)
	        * User chooses the inspection layout:
		        * Mark defects only
		        * Submit all inspection items
	        * Depending on layout selected the user will see the mobile preview in the correct layout
        * Task: Manual inspection form edit. If the user wants to change the form he can click on edit form. (mocked for now)  
        * Task: Publish inspection form and return to the Activity selection. When the user confirms the template is ok the agent will show that it's publishing the template and return to the initial 
    *   **Activity: Get all my drivers ready for mobile inspections**
        *   *Tasks:* Review AI-audited list of drivers showing their readiness status -> Click to edit drivers flagged as "Needs Attention" -> Add missing email addresses -> Enable mobile access via a toggle -> Assign drivers to one or more vehicle groups -> Save changes -> Add new drivers if needed.
    *   **Activity: Roll out training and monitor activation**
        *   *Tasks:* Review the AI-drafted training email and downloadable PDF guide -> Edit the email subject and body directly in the browser -> Use an AI prompt to ask for stylistic changes to the email -> Send the training materials to all drivers with one click -> Optionally, skip the email step and handle training manually -> View the activation dashboard to see which drivers have logged in.
